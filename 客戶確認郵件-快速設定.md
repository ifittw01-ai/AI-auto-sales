# 客戶確認郵件 - 快速設定

## ⚡ 3 分鐘完成設定

### 第 1 步：更新 Google Apps Script
```
1. 前往 https://script.google.com
2. 開啟你的專案
3. 複製 google-apps-script-updated.js 的全部內容
4. 貼上到 Code.gs（替換原有代碼）
5. 儲存 → 部署 → 管理部署作業 → 編輯 → 新版本 → 部署
```

### 第 2 步：測試
```
1. 前往你的網站
2. 填寫表單並提交
3. 檢查兩個郵箱：
   ✅ 推廣人員郵箱（通知郵件）
   ✅ 客戶郵箱（確認郵件）
```

## 🎯 新功能說明

✅ **推廣人員通知**（原有功能）
- 包含完整客戶資料
- 包含推廣代碼
- 包含聯繫方式

✅ **客戶確認郵件**（新功能）
- 感謝客戶報名
- 顯示選擇的評估地區
- 提供官方社群連結
- 友善的歡迎訊息
